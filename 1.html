Hypertext markup language (HTML) is the standard markup language for creating wed pages and wed applicaitions. Web browsers receive HTML documents from a web server or from local strage and render them into multimedia wed pages. HTML describes the structure of a web page semantically and originallry included cues for the apperance of the document

HTML elements are the building blocks of HTML pages. with HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page. It provides a means to create strucured documents by denoting structural semartics for text such as headings, paragrabs, lists, links, quotes and other items.
